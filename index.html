HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • h1->h6 - text headers with decreasing sizes as the number gets larger
  • ul - Unordered List; ol - Ordered List; li - List
  • br - Line Break; a - Anchor for hyperlinks
  • link - links files together
  • rel - specifies relation; href specifies reference location

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can style all elements of a certain type, or create a style class that you can apply to elements as you want to use it
  • text-align: aligns text to a position/direction within the elements boundaries
  • font-size: sets the size of the font in pixels
  • background-color: sets the color of the background, can be a semantic reference, an rbg value set, or a hex code
  • color: same as background-color, but for text
  • width: sets the width of an element, can be a pixel size or a percentage of available space
  • box-shadow: creates a drop shadow for an element, you can set height, width, color, blur, and spread radius

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: add all changes
  • git commit -m "": commit changes to current branch with a message
  • git pull [source] [branch]: pulls latest version of a branch from specified source
  • git push [source] [branch]: pushes latest version of a branch to a specified source

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions are blocks of code that you can call to make the code within run.
  • Standard arrays in JavaScript can hold more than one data type
  • If statements check if a value or condition is true
  • For loops iterate through the code within the loop a number of time based on the condition given
